# Atualização do cliente

> **PUT** /customers/_\{uuid}_

Pode ser atualizada a seguinte informação:

- **Identificador do cliente** (customerId)
- **Primeiro nome** (firstName)
- **Último nome** (lastName)
- **Email** (email)
- **Número de telefone** (phoneNumber)

**1.** Atualizar o número de telefone do cliente.
