# Eliminar uma entidade associada ao cliente

> **DELETE** /customers/_\{uuid}_/entities/_\{clientId}_

Para remover a associação do cliente com uma entidade é necessário conhecer o _uuid_ do cliente e o _clientId_ da entidade.

import ClickShowIframe from "../../src/utils/ClickShowIframe";

<button className="btn-api btn-api--float-right"
	onClick={() =>
		ClickShowIframe(
			"customer-entities",
			"customer-entities/delete/customers/{uuid}/entities/{clientId}"
		)
	}
>
	<span>Consultar API &nbsp;&nbsp;&nbsp;{'>'}</span>
</button>
