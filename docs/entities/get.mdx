# Consultar entidade

> **GET** /entities/_\{clientId}_

O parâmetro **clientId** é o NIF da entidade.

**Nota:** Para aceder a esta operação não é necessário se autenticar com o _PayPay-ClientId_.

import ClickShowIframe from "../../src/utils/ClickShowIframe";
