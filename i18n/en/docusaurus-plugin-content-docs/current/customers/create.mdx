# Create customer

This operation allows you to register the customer in PayPay. It is recommended to use the `customerId` parameter to avoid duplicate registrations.

Creation of a customer with the following data:

- **Customer identifier:** 1
- **First name:** José
- **Last name::** Silva
- **Email:** jose.silva@email.com
- **Phone number:** 351#961234567

> **POST** /customers
