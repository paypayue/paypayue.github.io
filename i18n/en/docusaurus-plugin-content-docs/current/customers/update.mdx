# Customer update

> **PUT** /customers/_\{uuid}_

The following information can be updated:

- **Customer identifier** (customerId)
- **First name** (firstName)
- **Last name** (lastName)
- **Email** (email)
- **Phone number** (phoneNumber)

**1.** Update the phone number of the customer.
